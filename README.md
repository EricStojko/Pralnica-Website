# 🧺 Pralnica Koper - Uradna Spletna Stran

Uradna spletna stran za samopostrežno pralnico Speed Queen v Kopru. 

🌍 **V živo na:** [pralnica-koper.si](https://pralnica-koper.si)

## 🛠️ Tech Stack
Projekt je zgrajen z mislijo na maksimalno hitrost nalaganja (Performance) in SEO optimizacijo:
- **Frontend:** Vanilla HTML5, CSS3, JavaScript
- **Gostovanje (Hosting):** Google Firebase Hosting
- **Jeziki:** Slovenščina (privzeto), Angleščina, Nemščina, Italijanščina, Hrvaščina

## 🤖 AI-Assisted Development
Ta projekt je bil zgrajen z uporabo modernih pristopov "AI Pair Programminga" (Project IDX, AI Agenti). 

Kot inženir in produktni vodja sem AI uporabil kot pospeševalnik za generiranje osnovne strukture, prevodov in CSS-a, medtem ko sem osebno bdel nad arhitekturo, QA testiranjem, odpravljanjem specifičnih hroščev (npr. reševanje UTF-8 Mojibake konfliktov pri avtomatiziranih prevodih) in končnim deploymentom. Ta projekt prikazuje mojo sposobnost učinkovitega upravljanja AI orodij za hitro dostavo produkcijsko pripravljenih rešitev.

## 🚀 Razvoj in Lokalno Testiranje
Ker gre za statično spletno stran brez kompleksnega zalednega sistema (backend), ne potrebujete posebnih "build" orodij.

1. Klonirajte repozitorij:
   `git clone https://github.com/EricStojko/Pralnica-Website.git`
2. Odprite mapo v poljubnem urejevalniku kode (VS Code, Project IDX, Cursor).
3. Uporabite razširitev "Live Server" za lokalno predogledovanje strani.

## 📦 Deployment (Objava)
Projekt je povezan s Firebase Hostingom. Za objavo sprememb uporabite terminal:
`firebase deploy`
